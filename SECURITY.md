# Security Policy

## Reporting a Vulnerability
If you discover a security issue or accidental secret exposure, please report it directly to the repository owner.

## Key Safety
- Never commit `.env` files or real API keys.
- Always use `.env.example` with placeholders only.
- Rotate keys immediately if they are shared or committed.

## Data Handling
Only collect and use data in compliance with TikTok Research API policies and applicable laws.
